Here we have a list of common terms and concepts to start you off.

JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Variables can be thought of as named containers. You can place data into these containers and then refer to the data simply by naming the container. Before you use a variable in a JavaScript program, you must declare it.

Naming Conventions

Improves code readability and makes code maintenance easier

Functions

A function is a JavaScript procedureâ€”a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Methods

JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition. Property. Value.

Arguments

The arguments object is a local variable available within all functions. You can refer to a function's arguments within the function by using the arguments object.

Parameters

Arguments are Passed by Value. The parameters, in a function call, are the function's arguments. JavaScript arguments are passed by value: The function only gets to know the values, not the argument's locations.

Return

In JavaScript, return statements cease execution in a function and return a value to the caller; JavaScript functions require an explicit return statement for returning the result of an expression (the value) from a function.

Chaining Methods

Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Strings

A JavaScript string stores a series of characters like "John Doe". A string can be any text inside double or single quotes.

Booleans

A JavaScript Boolean represents one of two values: true or false.

Escape

A string will be chopped to if we include quotation marks. The solution to avoid this problem, is to use the \ escape character. The backslash escape character turns special characters into string characters.

Alert

Alert(message); message is an optional string of text you want to display in the alert dialog.

Comments

To create a single line comment in JavaScript, you place two slashes "//" in front of the code or text you wish to have the JavaScript interpreter ignore. When you place these two slashes, all text to the right of them will be ignored, until the next line.

jQuery

Query is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

Attributes

One dynamic property is the attributes property of DOM elements. This attributes property provides the names and values of every attribute on a given element!
